# Soapy SDR module for Analog Devices ADRV9361 SDR

## Installation instructions

```
git clone git@github.com:samiemostafavi/SoapyADRV9361.git
cd SoapyADRV9361
mkdir build
cd build
cmake ..
make
sudo make install
```

## Dependencies

- [SoapySDR](https://github.com/pothosware/SoapySDR)
